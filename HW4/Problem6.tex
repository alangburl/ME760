\end{itemize}

The unit normal vector is described by Equation \ref{eq:unit_normal}.
\begin{equation}
	\hat{n}=\frac{\nabla f}{\|\nabla f\|}
	\label{eq:unit_normal}
\end{equation}

\begin{itemize}
	\item Using Equation \ref{eq:unit_normal} the resulting vector.
	\begin{equation*}
	\boxed{

	\end{equation*}
